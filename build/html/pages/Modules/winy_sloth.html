class winy_sloth.WinySloth[source]

Bases: object

Description: A class used to represent WinySloth main object. The general behaviour of Winy Sloth is described in Documentation.

Attributes: strategies_folder_path : Path of the folder containing all strategies

strategies : List containing StrategyFile objects

mode : Variable specifiying WS mode of execution. In DEBUG mode, WS is executed during 40s only. In RUN mode, WS runs nominally.

init_return outputs the return of initialization. If init_return is equal to 1, it means a problem occurred during initialization and the program exits. If 0, init was OK and WS enters in main.

WinySloth__Main()[source]

Name : WinySloth__Main()

Parameters :

Description : Main function of a Winy_Sloth object. This function is going through each strategy in the strategy list. For each strategy, the master’s position in the strategy file is compared with the master’s position given from the CEP. In case of difference, the position of each slave is updated copying the master’s position. If a position changes or if any error occurs, an email is sent.
